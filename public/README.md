# Depósito Fácil - Imagem do Hero (hero_banner.png)

Olá! Para exibir exatamente a imagem que você enviou no cabeçalho (Hero) da Landing Page, siga os passos abaixo:

1. Salve a sua imagem PNG sem fundo com o nome exato: **hero_banner.png**
2. Arraste e solte o arquivo **hero_banner.png** diretamente nesta pasta `public` no gerenciador de arquivos do editor de código do AI Studio.
3. Assim que o arquivo for carregado para `/public/hero_banner.png`, o sistema irá carregá-lo automaticamente no topo da Landing Page!

Enquanto a imagem não for carregada, o sistema exibirá uma composição 3D interativa de alta fidelidade desenhada via CSS.
